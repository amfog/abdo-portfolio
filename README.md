# Abdelrahman Mohamed Ahmed — Portfolio

Personal portfolio and product showcase.
Live: https://abdofog.vercel.app
Repo: https://github.com/amfog/abdo-portfolio

## Setup
```bash
npm install
npm run dev
```
Open http://localhost:3000

## Deploy
```bash
git push origin main
```
Vercel auto-deploys on push to main.

## Update Content
All site content is in `data/index.ts`.
Edit products, case studies, timeline, and stats there.
