# Project Overview — Abdelrahman Mohamed Ahmed Portfolio

## Project Name & Description

A personal portfolio website for **Abdelrahman Mohamed Ahmed** — Founder, Operations Architect, and Product Systems Leader. The site showcases 3+ years of building scalable workflow ecosystems, AI-assisted operations platforms, and esports infrastructures across MENA. Target roles: Product Operations, Program Management, Systems Leadership.

Live URL: deployed via Vercel (static export)

---

## Tech Stack

| Layer | Technology | Version |
|---|---|---|
| Framework | Next.js | 16.2.3 |

## Design System

### Colors
```
Background:  #0d0d20  (deep navy-black)
Primary:     #4f75ff  (electric blue)
Secondary:   #7b5cf6  (purple)
Text:        #ffffff  (white)
Muted text:  rgba(255,255,255, 0.40–0.70)
```

### CSS Custom Properties
```css
--primary:    #4f75ff
--secondary:  #7b5cf6
--bg-base:    #0d0d20
```

### Font
- **Geist** — loaded via `next/font/google`, variable `--font-geist`

### Glass Card Variants
```css
.glass        /* 3% white bg, 6% white border, blur 6px */
.glass-card   /* 8% white bg, 12% white border, blur 8px */
.glass-strong /* 5% white bg, 8% white border, blur 10px */
```

### Glow Utilities
```css
.glow-primary    /* blue glow box-shadow */
.glow-secondary  /* purple glow box-shadow */
```

### Gradient Text
```css
.gradient-text          /* white fade */
.gradient-text-primary  /* blue → purple */
.gradient-text-brand    /* blue → purple (alias) */
```

### Layout Container
```css
.container {
  max-width: 1200px;
  margin: auto;
  padding: 2rem (mobile) → 3rem (md) → 4rem (xl)
}
```

### Animations
- **Framer Motion** stagger containers on hero load
- `whileInView` reveal animations on all sections (once: true)
- `animate-pulse` on status indicator dot
- `.animate-fade-in-up` CSS keyframe utility

## Deployment

The site is configured for **static export** (`output: 'export'` in `next.config.ts`), deployed to **Vercel**.

```bash
# Build static export
npm run build
# Output goes to /out directory

# Deploy via Vercel CLI
vercel deploy --prod
```

GitHub Pages deployment script also present (`gh-pages` package in devDependencies).
